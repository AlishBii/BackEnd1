## About the project
Using this web application you can place your order for 3d printing. All orders are available in main page. Also, it is possible to see detailed information about specific order. 

### Built With

* Spring
* Spring Boot
* Spring MVC
* PostgreSQL
* REST API
* Swagger
* Bootstrap
* CSS
* HTML
